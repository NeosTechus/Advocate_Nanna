Setup Instructions
  1. Go to google.com/alerts
  2. Sign in with your Gmail account
  3. Paste each query from above into the search box
  4. Click "Show options"
  5. Set How often: "As-it-happens" (recommended) or "Once a day"
  6. Set Sources: "Automatic"
  7. Set Language: "English"
  8. Set Region: "India"
  9. Set How many: "All results"
  10. Set Deliver to: Your email
  11. Click "Create Alert"
  12. Repeat for all 12 queries above
